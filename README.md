## Projetos em destaque

### food4

Aplicação para conectar supermercados a cozinhas solidárias.

- **Stack:** C#, ASP.NET Core, Entity Framework Core, JWT, Swagger/OpenAPI, SQLite, SQL Server, JavaScript, HTML e CSS.
- **Destaques:** API com autenticação, controllers, models, migrations, camada de serviços e frontend web.
- **Repositório:** [antonioreal97/food4](https://github.com/antonioreal97/food4)

### RouteX

Sistema de gestão de frota de veículos com backend, frontend e infraestrutura em containers.

- **Stack:** Node.js, Express, React, Socket.IO, MongoDB, SQLite, Docker e Docker Compose.
- **Destaques:** comunicação em tempo real, dashboard, mapa, alertas e arquitetura separada entre frontend e backend.
- **Repositório:** [antonioreal97/RouteX](https://github.com/antonioreal97/RouteX)

### dash.notria

Dashboards para análise de dados da Ceasa.

- **Stack:** Python, Streamlit, Pandas, Plotly e OpenPyXL.
- **Destaques:** leitura de planilhas, visualizações interativas e dashboards por localidade.
- **Repositório:** [antonioreal97/dash.notria](https://github.com/antonioreal97/dash.notria)

### dashboard_matriz

Dashboard em Python para análise de matriz avaliativa.

- **Stack:** Python, Streamlit, Pandas, Plotly e OpenPyXL.
- **Destaques:** transformação de dados, filtros e visualização analítica.
- **Repositório:** [antonioreal97/dashboard_matriz](https://github.com/antonioreal97/dashboard_matriz)

### MLXp

Chat interativo com personalidades virtuais personalizáveis.

- **Stack:** JavaScript, Node.js, Express, Axios, Bcrypt, Cookie Parser, CSRF, Ollama e OpenAI API.
- **Destaques:** autenticação, interface web, integração com modelos de linguagem e gerenciamento de personas.
- **Repositório:** [antonioreal97/MLXp](https://github.com/antonioreal97/MLXp)

### Proj_ML

Coleção de projetos de machine learning, MLOps e deep learning.

- **Stack:** Python, Jupyter Notebook, Pandas, NumPy, FastAPI, Pydantic, XGBoost e Joblib.
- **Destaques:** EDA, preprocessamento, treinamento de modelos, API de predição e experimentos com imagens.
- **Repositório:** [antonioreal97/Proj_ML](https://github.com/antonioreal97/Proj_ML)

### RPG_game.beta

Jogo RPG desenvolvido em Python.

- **Stack:** Python, Pygame e PyMongo.
- **Destaques:** câmera, inventário, inimigos, NPCs, multiplayer, servidor e sistema de itens.
- **Repositório:** [antonioreal97/RPG_game.beta](https://github.com/antonioreal97/RPG_game.beta)

### notria

Site institucional para consultoria.

- **Stack:** HTML, CSS e JavaScript.
- **Destaques:** páginas institucionais, estilos modulares, animações e assets visuais.
- **Repositório:** [antonioreal97/notria](https://github.com/antonioreal97/notria)

### Rascunhos

Experimentos e estudos com notebooks, visão computacional e automação.

- **Stack:** Python, Jupyter Notebook, OpenCV, TensorFlow/Keras, NumPy, cvzone e pynput.
- **Destaques:** reconhecimento de expressão facial, redução de dimensão, classificação de imagens e apresentação com gestos.
- **Repositório:** [antonioreal97/Rascunhos](https://github.com/antonioreal97/Rascunhos)
